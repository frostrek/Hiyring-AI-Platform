# 📚 Technical & AI Documentation

This directory houses the technical blueprints and integration guides for the AI Hiring Platform.

## Contents

- `ai-models/`: Documentation on bias reduction, tone analysis, and natural language processing models.
- `video-streaming/`: Infrastructure documentation for handling high-volume concurrent video uploads and transcodings on AWS.
- `api-reference/`: Integration guides for hooking Hiyring into existing ATS (Applicant Tracking Systems).
